# Product Overview

This is a **Django-based Attendance Management System** — a web application for tracking and managing attendance records.

The project is in early development. The core app (`core`) is the primary Django app where all attendance-related features will be built. The UI uses a clean, minimal design with Tailwind CSS and Alpine.js for interactivity.

## Target Users
- Administrators managing attendance records
- Users (e.g., students, employees) who need to check in or view their attendance

## Key Goals
- Track attendance events (check-ins, absences, etc.)
- Provide an admin interface for managing records
- Deliver a responsive, accessible UI
